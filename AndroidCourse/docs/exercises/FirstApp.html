First App

Objectives of exercise:

  • create an Android project
  • run app on virtual device
  • run app on real device

Create an Android project; this can be done from a command line - see Android Developer Guide for more details; we are going to use Eclipse, as it's easier!

  • File | New | Android Application Project
  • supply application name, package name
  • minimum required SDK: API 8
  • target SDK and compile with: highest available
  • theme: one of the Holo themes
  • accept defaults for all other options in wizard
Configure a virtual device:
  • Window | Android Virtual Device Manager
  • choose one of the templates in Device Definitions or create one from scratch using Android Virtual Devices
Run your app on virtual device:
  • right-click on project name in Project Explorer, and choose Run As | Android application
  • choose emulator as the target; note that the emulator is quite big, and may take a few minutes to start up!
Your app should say "Hello world!"

Note: use the Esc key as the back button.

If you have an Android device, close down the emulator, and run your app on your own device.

  • Plug in your device to your development machine with a USB cable. If you're developing on Windows, you might need to install the appropriate USB driver for your device.
  • Enable USB debugging on your device.
For more information, see Running Your App in the Android Developer Guide.

If you are going to continue using the emulator, it's best to leave it running, as it can take some time to start up.

See if you can find out where the string Hello world! comes from, and change it.
